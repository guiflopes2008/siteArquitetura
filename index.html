Sistema operacional

O QUE É UM SISTEMA OPERACIONAL?

Fundamentalmente, um sistema operacional é um software, que pode ser o Linux, Windows, Android, macOS, UNIX, entre outros. No entanto, ele não resume aquilo que seus olhos conseguem ver ou ao que você consegue interagir. Em outras palavras, é um programa que conversa diretamente com o hardware da sua máquina. O sistema operacional assegura que os programas funcionem corretamente.

fonte

PRA QUE SERVE?

Um sistema operacional é o software essencial que gerencia o hardware de um computador, permitindo a execução de programas e aplicativos

fonte

CURIOSIDADE

Em novembro de 1985, a Microsoft lançou o Windows 1.0 oficialmente, como sua primeira tentativa de trazer para o público um sistema interativo. Ele era bastante limitado, vindo apenas com o bloco de notas, um relógio, a calculadora, o velho e antigo Paint, um joguinho chamado Reversi, um editor para cartões, o Microsoft Writer e outros programas de computação. Um de seus destaques é que ele contava com muitas cores, em um visual novo e chamativo. Ele também se destacou por ser o primeiro a utilizar o mouse para facilitar o uso e também com suporte a atalhos na tela pelos ícones e janelas. Porém, mesmo com essas melhorias, a Microsoft só declarou o Windows como um sistema operacional de fato com a chegada do Windows NT em 1993.

Driver

O QUE É UM DRIVER

Um driver é um componente de software que permite que o sistema operacional e um dispositivo se comuniquem. Por exemplo, quando um aplicativo precisa ler dados de um dispositivo, ele chama uma função implementada pelo sistema operacional. Em seguida, o sistema operacional chama uma função implementada pelo driver. O driver, geralmente desenvolvido pelo fabricante do dispositivo, sabe como se comunicar com o hardware do dispositivo para obter os dados. Depois que o driver obtém os dados, ele os devolve ao sistema operacional, que os devolve ao aplicativo.

fonte

PRA QUE SERVE?

De forma simples, o driver é um pequeno programa que permite que o sistema operacional do computador possa se comunicar com o dispositivo conectado a ele. Basicamente faz com que um mouse, uma impressora e outros componentes possam ser interpretados pelo sistema e realizem suas ações da forma correta.

SOFTWARE PROPRIETÁRIO

O QUE É

Software proprietário é aquele que pertence a uma pessoa ou empresa e pode ter restrições para uso, modificação e distribuição. O código-fonte do software proprietário não é disponibilizado ao público. Isso significa que os usuários não podem ver, modificar ou compartilhar o código.

O software proprietário é vendido comercialmente e os usuários devem adquirir uma licença para usar. Os termos da licença podem limitar o uso do software, como limitar o número de instalações ou tipos de dispositivos que podem ser usados.

fonte

PRA QUE SERVE?

Software proprietário é um software para computadores licenciado com direitos exclusivos para o produtor. O software pode ser protegido por patentes, direitos de autor e restrições para ser usado em outros países.

TIPOS DE LICENÇA DE SOFTWARE

O QUE É

Uma licença é quando o desenvolvedor do sistema autoriza ou limita certas ações. As licenças podem ser gratuitas, mas normalmente são pagas.

fonte

PRA QUE SERVE?

funciona como um tipo de contrato estabelecido entre o fabricante (desenvolvedor) e a pessoa ou empresa que o usará.

fonte

CURIOSIDADES

Qual é o tempo médio de validade para as licenças?

Toda licença de software possui uma data de validade, exceto a de uso perpétuo. Nesta, a permissão é anulada sempre que alguém ferir alguma cláusula do contrato, desrespeitando os termos de utilização. Algumas licenças precisam ser renovadas a cada atualização do sistema para que o usuário tenha acesso às novas versões. Já outras possuem renovação automática. O mais importante aqui é descobrir o tipo de licença utilizado pelo fornecedor do seu software e ficar atento às mudanças realizadas nos termos de uso, além de verificar se está acessando as versões mais atualizadas do sistema. No setor de TI, há uma grande preocupação com o contrato de prestação de serviços e poucos gestores prestam atenção na licença de software, com os termos de uso, à qual estão sendo submetidos. Você, como profissional da área, tem o dever de mudar isso, conscientizando fornecedores e usuários sobre o assunto.

COPYRIGHT

O QUE É

Copyright é o direito do autor de reproduzir sua obra artística, artística ou científica. Na maioria das vezes, a palavra "copyright" vem acompanhada da frase "todos os direitos reservados", que significa que a obra está protegida por uma lei. Mas há três tipos de proteção para o copyright: "todos os direitos reservados", "alguns direitos reservados" e "domínio público". A expiração do copyright varia de acordo com a legislação. No Brasil, os direitos do autor podem durar por 70 anos depois que ele morre. Depois disso, a obra é considerada pública.

fonte

PRA QUE SERVE?

É a maneira mais recomendada de proteger os produtos, ideias e serviços de uma empresa. Com ele, é possível evitar pirataria e processar aqueles que copiam materiais de outras pessoas para ganhar dinheiro. Assim, oferece segurança aos produtores de conteúdo e ajuda a profissionalizar o mercado.

fonte

CURIOSIDADE

O copyright é realmente a primeira proteção que se tem a uma obra autoral, teve seu início na Inglaterra , a primeira lei de direitos autorais foi promulgada no Reino Unido em 1710, conhecida como Tratado de Anne, em referência à rainha Anne (1665-1714).

COPYLEFT

O QUE É

Copyleft é uma proteção de propriedade intelectual que permite a cópia, alteração e reprodução da obra, desde que a liberdade seja mantida em todas as versões modificadas e distribuídas. O Copyleft é uma licença que autoriza a cópia para permitir a livre circulação e o acesso. Ele surgiu para ajudar a aprender mais sobre programação de software.

fonte

PARA QUE SERVE?

Copyleft é um método geral para tornar um programa (ou outra obra) livre

fonte

CURIOSIDADE

O copyleft também não proíbe a venda da obra pelo autor, mas implica a liberdade de qualquer pessoa fazer a distribuição não comercial da obra.

creative commons

O QUE É

Creative Commons é uma licença que permite aos autores usarem seu trabalho de uma maneira específica. Em geral, conteúdo do tipo CC não requer permissão ou taxa de licenciamento. Obras em Creative Commons ainda são protegidas por direitos autorais. Os criadores criaram coisas novas para serem usadas. Criadores podem escolher como seus trabalhos serão usados ou revisitados, mesmo sob regime. A Creative Commons Office é uma organização sem fins lucrativos que permite o compartilhamento e uso de material criativo. O escritório desenvolve, apoia e administra a infraestrutura legal para a criatividade digital e a inovação. A organização tem muitos membros espalhados pelo mundo.

fonte

PRA QUE SERVE

O Creative Commons é uma organização mundial sem fins lucrativos que permite o compartilhamento e a reutilização da criatividade e do conhecimento por meio de ferramentas jurídicas gratuitas.
